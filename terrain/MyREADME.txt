Randomly generated height map is done
Terrain generation according to height map is done
Application of a single texture to terrain is done
Randomly placed simple textured 3D objects over terrain is done
Simple global light source is done (Sun)
Navigation using keyboard buttons is done

Additional Requirements:
Loading height map from file is done
Applying multiple textures to define different levels of the terrain is done (3 textures are used)
Use multiple light sources are done (sun, star and 1 moving with the camera)

Note:
Textures taken from local. You can install extension or use Avast Secure Browser
